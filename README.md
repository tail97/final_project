# 자연어 처리를 이용한 감성 일기장 

- 대상 : 몸과 마음이 힘든 MZ세대 
- 프로젝트 설명 : 몸과 마음이 지친 MZ세대를 위한 일기장입니다. 하루에 있었던 일들을 일기에 적으므로써 적은 글을 모델이 감정분석을 해서
                  나쁜 감정이 나오면 감정을 위로 할수 있는 컨텐츠를 추천, 긍정 감정이 나오면 감정을 더 시너지 있게 해주는 컨텐츠를 추천해주는 서비스입니다 .
